## はじめに

PyAim Cross-Platform Trackerは、FPS/TPSゲームのエイム精度を向上させるためのトレーニングツールです。

### 主な機能

- **2つのトレーニングモード**: Tracking（追いエイム）とFlicking（瞬間エイム）
- **デバイス対応**: マウスとゲームパッドの両方に対応
- **詳細な統計**: セッション結果の自動保存とグラフ表示
- **カスタマイズ**: 時間、ターゲット数、感度などを調整可能

---

## 起動方法

### Windows
```cmd
cd TrackingAim
.venv\Scripts\activate
python main.py
```

### macOS / Linux
```bash
cd TrackingAim
source .venv/bin/activate
python3 main.py
```

### VS Code
F5キーを押すとデバッグモードで起動します。

## ランチャー画面

アプリケーションを起動すると、最初にランチャー画面が表示されます。

### 画面構成

#### 上部
- **タイトル**: PyAim Tracker
- **デバイス情報**: 接続中のデバイス（マウス/ゲームパッド）

#### 中央
- **Trackingボタン**: 追いエイムモードを開始
- **Flickingボタン**: 瞬間エイムモードを開始

#### 設定スライダー
1. **感度**: ゲームパッドの感度（100-1000）
2. **デッドゾーン**: ゲームパッドのデッドゾーン（0.0-0.3）
3. **Tracking時間**: セッション時間（10-60秒）
4. **Flickingターゲット数**: ターゲット数（5-30個）

#### 下部ボタン
- **保存**: 現在の設定を保存
- **統計**: 統計画面を表示

### 操作方法
- **マウス**: クリックで選択
- **ゲームパッド**: 自動検出（接続時）
- **ESCキー**: アプリケーション終了

## トレーニングモード

### Trackingモード（追いエイム）

動くターゲットにカーソルを合わせ続けるトレーニングです。

#### 目的
- 移動するターゲットを追従する能力を鍛える
- 安定したエイムを維持する練習

#### 画面表示
- **残り時間**: 右上に表示
- **リアルタイムT0率**: 現在のターゲット捕捉率
- **ON TARGET**: ターゲットに命中中の表示

#### ターゲットの動き
- ランダムな方向転換（1.5-3.5秒間隔）
- 速度変化（基本速度の70-130%）
- 画面端での反射

#### 評価基準
- **T0率（Time on Target）**: ターゲットにカーソルが重なっていた時間の割合
  - S: 80%以上 - Excellent!
  - A: 60-79% - Great!
  - B: 40-59% - Good
  - C: 40%未満 - Keep practicing

#### リザルト画面
- T0率（アニメーション付き）
- 評価ランク
- 直近5セッションのグラフ
- **もう一度ボタン**: 同じ設定で再挑戦

### Flickingモード（瞬間エイム）

ランダムに出現するターゲットを素早くクリックするトレーニングです。

#### 目的
- 瞬間的なエイム精度を鍛える
- 反応速度を向上させる

#### 画面表示
- **進捗**: 現在のターゲット数/総数
- **Hits**: ヒット数
- **反応速度**: 直近の反応速度（ミリ秒）

#### ルール
- ターゲットをクリックすると次のターゲットが出現
- ミスしても次のターゲットへ進む
- 設定したターゲット数に到達したら終了

#### エフェクト
- **ヒット時**: 緑のパーティクル爆発
- **ミス時**: 赤のパーティクル

#### 評価基準
- **命中率**: ヒット数 / 総ターゲット数
- **平均反応速度**: 全ヒットの平均反応時間
- **最速反応速度**: 最も速かった反応時間

**ランク:**
- S: 命中率90%以上 & 平均250ms未満 - Amazing!
- A: 命中率70%以上 - Great!
- B: 命中率50%以上 - Good
- C: 命中率50%未満 - Keep practicing

#### リザルト画面
- 命中率（アニメーション付き）
- 平均反応速度
- 最速反応速度
- 評価ランク
- 直近5セッションのグラフ

### 共通操作
- **ESCキー**: セッション中断 / ランチャーに戻る
- **マウスクリック**: ターゲット選択（Flickingモード）

## 統計画面

過去のトレーニング結果を確認できます。

### 表示内容

#### Tracking Mode
- セッション数
- 平均T0率
- 最高T0率
- 直近10セッションのグラフ（緑色）

#### Flicking Mode
- セッション数
- 平均命中率
- 最高命中率
- 平均反応速度
- 直近10セッションのグラフ（赤色）

### グラフの見方
- **横軸**: セッション番号（古い→新しい）
- **縦軸**: スコア（T0率 or 命中率）
- **線**: スコアの推移
- **点**: 各セッションのスコア

### 操作
- **戻るボタン**: ランチャー画面に戻る
- **ESCキー**: ランチャー画面に戻る

## 設定のカスタマイズ

### 入力設定（ゲームパッド）

#### 感度（Sensitivity）
- **範囲**: 100-1000
- **推奨**: 300-500
- **効果**: 高いほどカーソルの移動速度が速い

#### デッドゾーン（Deadzone）
- **範囲**: 0.0-0.3
- **推奨**: 0.05-0.15
- **効果**: スティックの遊びの範囲。高いほど微小な入力を無視

### トレーニング設定

#### Tracking時間
- **範囲**: 10-60秒
- **推奨初心者**: 15-20秒
- **推奨上級者**: 30-60秒

#### Flickingターゲット数
- **範囲**: 5-30個
- **推奨初心者**: 5-10個
- **推奨上級者**: 15-30個

### 設定の保存

1. スライダーで値を調整
2. 「保存」ボタンをクリック
3. `profiles/default.json`に保存される
4. 次回起動時に自動で読み込まれる

## データ管理

### 保存場所

```
TrackingAim/
├── data/
│   └── sessions/
│       ├── tracking.csv    # Trackingモードの履歴
│       └── flicking.csv    # Flickingモードの履歴
└── profiles/
    └── default.json        # 設定ファイル
```

### CSVファイル形式

#### tracking.csv
```csv
timestamp,mode,t0_rate,duration
2026-01-18T12:00:00,tracking,65.4,30.0
```

#### flicking.csv
```csv
timestamp,mode,accuracy,avg_reaction_ms,min_reaction_ms,hits,total
2026-01-18T12:05:00,flicking,80.0,245,180,8,10
```

### データのバックアップ

定期的に`data`フォルダと`profiles`フォルダをバックアップすることを推奨します。

## よくある質問

### Q: ゲームパッドが認識されない
**A:** 
1. ゲームパッドを接続し直す
2. アプリケーションを再起動
3. Windowsの場合、デバイスマネージャーで認識を確認

### Q: フォントが文字化けする
**A:** 
- Windows: MS Gothic、Meiryo、Yu Gothicのいずれかをインストール
- macOS: Hiragino Sansは標準搭載
- Linux: `sudo apt install fonts-noto-cjk`

### Q: 統計が表示されない
**A:** 
最低1回はトレーニングを完了させる必要があります。

### Q: 設定が保存されない
**A:** 
`profiles`フォルダの書き込み権限を確認してください。

### Q: 画面がカクつく
**A:** 
1. 他のアプリケーションを閉じる
2. グラフィックドライバーを更新
3. 設定で時間/ターゲット数を減らす

### Q: データをリセットしたい
**A:** 
`data/sessions`フォルダ内のCSVファイルを削除してください。

### Q: 複数のプロファイルを使いたい
**A:** 
現在は1つのプロファイル（default.json）のみ対応しています。

## トレーニングのコツ

### Trackingモード
1. **最初は短時間で**: 15秒から始めて徐々に延長
2. **リラックス**: 肩の力を抜いて
3. **予測**: ターゲットの動きを予測する
4. **目標**: まずはT0率50%を目指す

### Flickingモード
1. **焦らない**: 正確性を優先
2. **リズム**: 一定のリズムでクリック
3. **視野**: 次のターゲット位置を素早く把握
4. **目標**: まずは命中率70%を目指す

### 上達のために
- **毎日練習**: 短時間でも継続が重要
- **統計を確認**: 成長を可視化
- **休憩**: 疲れたら休憩を取る
- **実戦**: ゲームでの実践も重要
